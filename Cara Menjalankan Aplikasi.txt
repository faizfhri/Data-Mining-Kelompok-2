A. Link Deployment
Aplikasi dapat langsung digunakan dengan mengakses link berikut:
https://data-mining-kelompok2.streamlit.app/

B. Local Server
1. Buka command prompt, pastikan sudah berada dalam direktori tempat file txt ini berada

2. Install semua library yang diperlukan dengan perintah:
pip install -r requirements.txt

3. Jalankan aplikasi streamlit dengan perintah:
streamlit run app.py
